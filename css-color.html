Propriedade CSS: color

A propriedade color é usada para definir a cor do texto de um elemento.

A cor pode ser especificada por um nome de cor válido (como 'red'), um valor HEX (como '#ff0000') ou um valor RGB (como 'rgb(255, 0, 0)').

Exemplo de uso:

p.meu-texto {
    color: #3b82f6; /* Define a cor azul */
}
