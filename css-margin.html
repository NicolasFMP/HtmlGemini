Propriedade CSS: margin

A propriedade margin no CSS é usada para criar espaço em volta dos elementos, do lado de fora das bordas (borders).

Você pode definir a margem para todos os lados de uma vez, ou especificar margens individuais para o topo (top), direita (right), baixo (bottom) e esquerda (left).

Exemplo de uso:

.minha-caixa {
    margin-top: 20px;
    margin-bottom: 20px;
    
    /* Ou usando a forma resumida (shorthand): */
    /* margin: cima direita baixo esquerda */
    margin: 20px 10px 30px 15px;
}
Esta caixa azul tem uma margin definida no CSS para não encostar nos outros textos!
