A Tag <a> (Âncora/Links)

A tag <a>, ou anchor, é usada para criar links (hiperlinks) que conectam uma página da web a outra, ou a diferentes partes da mesma página.

O atributo mais importante desta tag é o href, que indica o destino do link. Outro atributo muito útil é o target="_blank", que usamos no nosso site para abrir o link em uma nova aba.

Exemplo de uso:

<!-- Link simples para outro site -->
<a href="https://www.google.com">Visite o Google</a>

<!-- Link abrindo em uma nova aba -->
<a href="contato.html" target="_blank">Fale Conosco</a>
