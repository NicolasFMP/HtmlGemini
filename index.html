Tutorial: O que é HTML?

O HTML (HyperText Markup Language) é a linguagem de marcação padrão para criar páginas web. Ele descreve a estrutura de uma página da Web usando marcação, consistindo em uma série de elementos (tags).

Veja um exemplo da estrutura básica de um documento HTML:

<!DOCTYPE html>
<html>
<head>
    <title>Título da Página</title>
</head>
<body>
    <h1>Meu Primeiro Cabeçalho</h1>
    <p>Meu primeiro parágrafo.</p>
</body>
</html>

As tags geralmente vêm em pares, como <p> e </p>. A primeira é a tag de abertura e a segunda é a tag de fechamento.

Tutorial: O que é CSS?

O CSS (Cascading Style Sheets) é usado para formatar o layout de uma página da web. Com o CSS, você pode controlar a cor, a fonte, o tamanho do texto, o espaçamento entre elementos, como os elementos são posicionados, imagens de fundo e muito mais.

Veja como podemos estilizar a tag <h1> do exemplo anterior:

h1 {
    color: blue;
    font-size: 24px;
    text-align: center;
}

Neste exemplo, h1 é o seletor. Tudo dentro das chaves {} é a declaração de estilo, composta por propriedades (como color) e valores (como blue).
